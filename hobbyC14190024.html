MY HOBBY

Hobby saya lebih santai dan tidak berhubungan apapun dengan pelajaran ataupun kuliah. Yaitu bermain game terutama game dengan genre Multiplayer Online Battle Arena atau biasa dikenal dengan MOBA dan juga game yang memiliki sistem gacha. Gacha sendiri adalah sistem mirip dengan lootbox di dalam game yaitu cara untuk mendapatkan karakter atau item tertentu dalam game dengan chance yang random. Menurut saya sistem ini menyenangkan karena dapat mengukur seberapa beruntung diri saya.

Contoh game dari masing-masing genre adalah Dota 2 yaitu game MOBA yang cukup terkenal dan saya senang bermain game ini diwaktu luang dengan teman-teman saya. Kemudian untuk game yang memiliiki gacha saya memiliki bebepa contoh game yang pertama adalah Fate/Grand Order saya sudah cukup lama bermain game ini dan game ini adalah game dengan genre gacha pertama yang saya mainkan. Selanjutnya game gacha lainnya adalah Blue Archive, Genshin Impact,dan Arknights ini hanyalah beberapa contoh game gacha dan kebanyakan selain sistem gacha, yang membuat saya tertarik adalah gameplay dari game
